# 🚀 SpringBoot + Vue 前后端分离项目实战

---

## 1. 总体目标 

前台为图书馆部分，用户可查阅书籍并发布个人笔记，后台为用户个人中心，基于角色的不同用户的权限不同，管理员可对用户角色进行修改等
